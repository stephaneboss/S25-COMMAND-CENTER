# Contexte Vietnam — Contacts

Dossier de fiches contact pour le contexte Vietnam. Structure non destructive, templates uniquement.

## Règles du dossier

1. **Aucun secret ici.** Jamais de seed phrase, clé privée, mot de passe, API key. Adresses publiques et identifiants internes seulement.
2. **Rien d'inventé.** Chaque champ reste `À REMPLIR` tant que l'information n'est pas confirmée par la personne concernée.
3. **Consentement d'abord.** Une fiche ne se remplit qu'avec le consentement explicite et documenté de la personne (date + portée).
4. **Vérifications obligatoires** avant toute interaction financière : identité et majorité légale vérifiées et documentées (méthode + date). Statut par défaut : `NON VÉRIFIÉ`.
5. **Identifiants internes** (ex. `VN-001`) pour référencer la personne dans les autres systèmes S25, jamais son nom complet.
